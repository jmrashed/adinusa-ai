How it works

1

Type a prompt

Describe what you want in plain English inside the VS Code chat panel.

2

Agent reasons

The backend agent loop plans steps, calls tools, and iterates until the task is done.

3

Apply changes

Review the inline action card and hit Apply — files are written and opened instantly.

Supported LLM Providers

Use any provider — switch anytime with Ctrl+Shift+P

GLM-4 (Zhipu) OpenAI GPT-4 Anthropic Claude Google Gemini Ollama (local)

Quick Start

1. Clone & configure

git clone https://github.com/jmrashed/adinusa-ai.git
cd adinusa-ai
cp backend/.env.example backend/.env

2. Install & start backend

bash scripts/install.sh
cd backend && npm run dev

3. Launch extension

Open adinusa-ai/ in VS Code → Press F5
